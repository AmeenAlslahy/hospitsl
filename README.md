# نظام إدارة المستشفيات

## وصف المشروع
نظام إدارة المستشفيات هو نظام شامل ومتكامل لإدارة العمليات اليومية في المستشفيات. يوفر النظام حلولاً متقدمة لإدارة المواعيد الطبية، السجلات الطبية، إدارة الأطباء والمرضى، وغيرها من الوظائف الأساسية في المستشفيات.

## المميزات الرئيسية
- **حجز المواعيد عبر الإنترنت**: إمكانية حجز المواعيد مع الأطباء بسهولة وسرعة
- **إدارة السجلات الطبية**: حفظ وإدارة جميع السجلات الطبية للمرضى
- **إدارة الأطباء والمرضى**: إضافة وتحديث بيانات الأطباء والمرضى
- **نظام الرواتب والوظائف**: إدارة الوظائف والطلبات عليها
- **التبرع بالدم**: إدارة عمليات التبرع بالدم
- **التقارير والإحصائيات**: تقارير مفصلة عن الأداء والإحصائيات
- **واجهة متجاوبة**: يعمل النظام على جميع الأجهزة (حواسيب، هواتف، أجهزة لوحية)
- **أمان وحماية**: تشفير البيانات وحماية خصوصية المرضى

## المتطلبات
لتشغيل هذا النظام، تحتاج إلى:
- **خادم الويب**: Apache أو Nginx
- **PHP**: إصدار 7.4 أو أحدث
- **قاعدة البيانات**: MySQL أو MariaDB
- **بيئة التطوير**: XAMPP أو WAMP أو LAMP (موصى به XAMPP للويندوز)

## التثبيت والإعداد
1. **تحميل المشروع**:
   - قم بتحميل ملفات المشروع إلى مجلد `htdocs` في XAMPP (أو المجلد المناسب لخادم الويب)

2. **إعداد قاعدة البيانات**:
   - قم بإنشاء قاعدة بيانات جديدة في MySQL
   - قم باستيراد ملف `hospital_management_system (2).sql` إلى قاعدة البيانات

3. **تكوين الإعدادات**:
   - افتح ملف `includes/config.php`
   - قم بتحديث إعدادات قاعدة البيانات (DB_HOST, DB_NAME, DB_USER, DB_PASS)
   - تأكد من صحة الإعدادات الأخرى مثل SITE_URL وUPLOADS_DIR

4. **تشغيل النظام**:
   - شغل خادم Apache وMySQL في XAMPP
   - افتح المتصفح واذهب إلى `http://localhost/hospital-system`

```
├── doctor/             # لوحة تحكم الطبيب

├── functions/          # الدوال المساعدة
├── uploads/            # الملفات المرفوعة

├── login.php           # صفحة تسجيل الدخول
├── register.php        # صفحة التسجيل
└── README.md           # هذا الملف
```

## الاستخدام
- **المدير**: يمكنه إدارة الأطباء، المرضى، المواعيد، والتقارير
- **الطبيب**: يمكنه عرض المواعيد، إضافة السجلات الطبية، وكتابة الوصفات
- **المريض**: يمكنه حجز المواعيد، عرض سجلاته الطبية، وإدارة حسابه
